# restore_npy_instructions.txt

본 프로젝트는 GitHub 저장소 용량 제한 및 보안 정책으로 인해 .npy 파일을 포함하지 않습니다.

따라서 프로젝트 실행 전, 아래 명령어를 통해 .npy 파일을 자동으로 복원해 주세요:

    python3 restore_npy.py

해당 스크립트는 다음 경로에 아래 파일들을 생성합니다:

경로: catkin_ws/src/localization_correction/cfg/

- myungsin_T_3d_to_2d.npy  
- renaissance_T_3d_to_2d.npy  

이후 별도 설정 없이 프로젝트를 실행하실 수 있습니다.
